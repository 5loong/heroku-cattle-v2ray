# Cattle 的 V2Ray 科学上网服务

在OpenShift、Heroku、IBM Bluemix 上部署 V2Ray 服务。

## Files

- `Procfile` - for Heroku
- `manifest.yml` - for IBM Bluemix

也支持 OpenShift ，使用它内置的 Builder
OpenShift 不要求额外的文件。
